Checking for Boost

Create an m4 directory in the root of your project and copy "boost.m4" into there. Add the following lines to your configure.ac file

AC_CONFIG_MACRO_DIR([m4])
# ...
AC_PROG_CXX
AC_PROG_LIBTOOL
# ...
BOOST_REQUIRE
In your root Makfile.am file write the following line:

ACLOCAL_AMFLAGS=-I m4
