--- layout: post title: Specifying install directories of QT Libraries and Headers date: '2014-03-15T14:31:00.000-07:00' author: James Hook tags: modified_time: '2014-03-15T14:32:11.149-07:00' blogger_id: tag:blogger.com,1999:blog-5482187274444893467.post-2610762241582775796 blogger_orig_url: http://computer-says-yes.blogspot.com/2014/03/specifying-install-directories-of-qt.html --- Add a few lines to your .pro file



headers.files = $$HEADERS
target.path = /usr/local/lib/yourproj
headers.path = /usr/local/include/yourproj

INSTALLS += headers target
