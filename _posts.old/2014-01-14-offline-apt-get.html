--- layout: post title: Offline apt-get date: '2014-01-14T03:27:00.000-08:00' author: James Hook tags: modified_time: '2014-01-14T03:32:14.000-08:00' blogger_id: tag:blogger.com,1999:blog-5482187274444893467.post-2059552426651133712 blogger_orig_url: http://computer-says-yes.blogspot.com/2014/01/offline-apt-get.html visible: 1 --- If you ever need to use apt-get on a machine that isn't connected to the Internet then follow these instructions

1) Create an identical machine to your offline machine (same architecture, version of debian, ubuntu etc.)

2) Clear out the package cache - this is so we get the sub-packages just for the desired package - not for every package you have ever downloaded. 
sudo rm /var/cache/apt/archives/*.deb
3) Issue a command to download but not install the desired package 
sudo apt-get --download-only install
4) Copy .deb files to a CD or USB stick (assume usb stick is empty) 
cp /var/cache/apt/archives/*.deb /media/usbstick/
5) insert usb stick into offline machine

6) install packages 
sudo dpkg -i /media/usbstick/*.deb
